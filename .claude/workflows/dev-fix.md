When fixing bugs or implementing features:

1. Inspect the relevant code first.
2. Explain the plan briefly.
3. Implement the change with minimal edits.
4. Run:
   npm run lint
   npm run test
   npm run build
5. If any command fails:
   - identify the cause
   - fix the code
   - rerun the command
6. Confirm that the application still works.

Never skip verification steps.
